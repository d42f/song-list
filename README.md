# Song List App

React song browser with infinite scroll, search and favorites.

## Stack

- React 18 + TypeScript
- TanStack Query — data fetching, infinite scroll, optimistic updates
- CSS Modules
- Vitest + React Testing Library

## Getting started

```bash
npm install
npm run dev                  # copies api/db.json.example → api/db.json on first run
```

App runs at `http://localhost:5173`, API at `http://localhost:3004`.

## Scripts

```bash
npm run dev      # development
npm test         # run tests
npm run build    # production build → public/
npm start        # serve production build
```
